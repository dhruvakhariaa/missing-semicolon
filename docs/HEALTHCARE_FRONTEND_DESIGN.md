# Healthcare Frontend Design Guide

> For Figma Design - Service Delivery Platform Hackathon

---

## Available Services & APIs

### 1. Departments
| Endpoint | Description |
|----------|-------------|
| `GET /departments` | List all hospital departments |
| `GET /departments/:id` | Get department details |

**Data Available:**
- Name, Code, Description, Icon
- Location (Building, Floor, Room)
- Contact info

---

### 2. Doctors
| Endpoint | Description |
|----------|-------------|
| `GET /doctors` | List all doctors (filterable) |
| `GET /doctors/:id` | Get doctor profile |
| `GET /doctors/:id/availability?date=YYYY-MM-DD` | Get available time slots |

**Data Available:**
- Name, Photo, Specialization
- Qualification, Experience (years)
- Consultation Fee (â‚¹)
- Rating, Total Reviews
- Available Days & Time Slots

---

### 3. Patients (Requires Login)
| Endpoint | Description |
|----------|-------------|
| `POST /patients` | Register patient profile |
| `GET /patients/me` | Get own profile |
| `PUT /patients/:id` | Update profile |

**Data Available:**
- Name, Email, Phone, DOB, Gender
- Blood Group, Address
- Emergency Contact
- Medical History (Allergies, Conditions)

---

### 4. Appointments (Requires Login)
| Endpoint | Description |
|----------|-------------|
| `POST /appointments` | Book new appointment |
| `GET /appointments` | List my appointments |
| `GET /appointments/:id` | Get appointment details |
| `PUT /appointments/:id` | Update appointment |
| `DELETE /appointments/:id` | Cancel appointment |

**Data Available:**
- Appointment Number (APT20260117xxxx)
- Doctor, Department info
- Date & Time Slot
- Status (scheduled/confirmed/completed/cancelled)
- Symptoms, Notes
- Fee, Payment Status
