# Scripts

Utility scripts for development and operations.

## Available Scripts

### `seed-data.js`
Seeds the database with demo data for presentation.
- Sample users (citizen, admin, provider)
- Sample appointments
- Sample farmers and advisories
- Sample complaints

### `generate-docs.js`
Generates API documentation from code comments.

### `health-check.js`
Checks health status of all services.

### `load-test.js`
Runs load tests to demonstrate scalability.
